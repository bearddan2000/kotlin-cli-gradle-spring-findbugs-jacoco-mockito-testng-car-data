# kotlin-cli-gradle-spring-findbugs-jacoco-mockito-testng-car-data

## Description
A POC for spring app using testng,
findbugs, jacoco, mockito framework.

## Tech stack
- kotlin
- gradle
  - spring
  - testng  
  - findbugs
  - jacoco
  - mockito

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
